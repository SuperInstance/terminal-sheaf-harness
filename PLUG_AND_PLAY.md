# PLUG_AND_PLAY — terminal-sheaf-harness

> *Integration guide.*


## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library |
| `alloc` | yes | Allocator support |

## Integration

Import `terminal_sheaf_harness` in your project to access the functionality.

```
use terminal_sheaf_harness::{};
```

## Compatibility

- **Rust edition**: 2021+
- **Targets**: All tier-1 Rust targets
